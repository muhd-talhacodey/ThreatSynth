# ThreatSynth ~ Turning Chaotic Data into Actionable Intelligence :) 

It has been hosted on Vercel, here is the link: 

ThreatSynth is a high-performance intelligence synthesis platform which was designed for cyber analysts who want to carry out analyzing pdfs with help of Notebook LLM. It bridges the gap between raw data (PDF/DOCX) and actionable insights by leveraging advanced generative AI to automate complex analysis for cyber analysts. The main reason behind this project was to ease difficulties for cyber analysts when analyzing different security incidents. 

So the main question is what does ThreatSynth has to offer at its core capabilitiies... 

First of all it has a smooth interactive UI, making it easier for Cyber Analysts to place their incident reports. 
It uses a PDF/Docx report which allows user to view the document in real-time experience. It even has a view bar from where a user can view the name of the file which has been uploaded. 


a) TELEMETRY READOUT
This is the first Nav bar for ThreatSynth which allows user to view their files in real-time for example if a Cyber Analyst is viewing the file to examine the incident report. It has an option to generate slides which allows the user to generate slides in real time to summarize the content in the pdf/docx file making it much easier for Cyber Analyst to find out the vulnerabilities in the report uploaded. 

b) THREAT GRAPH 
The second Nav bar for ThreatSynth uses mindmap in order to explain the vulnerabilities in short form to Cyber Analysts with key important words making it more easier to find threats in the report. 

c) BRIEFING 
The third Nav bar for ThreatSynth is used for making presentation with the option to generate slides which allows Cyber Analyst to explain the vulnerabilities in slides format whether it is in the workplace or in a specific competition. Slides will allow for deeper analysis with the incident report for Cyber Analysts making complicated searching for vulnerabilites easier on the specific user's fingertips. 

d) LIVE CVE
Last Nav bar helps Cyber Analysts to make their life easier by fetching out CVE in real time, making it way easier to find the issues in the report. It has a Unique Identification which is assigned with every vulnerability making it easier for security professionals to find flaws in the report. It uses a CVE ID to find any issues in the file. CVEs link directly to the NVD (National Vulnerability Database), which assigns a severity score (CVSS), this helps to fix those bugs first according to the severity of the file. 

ThreatSynth uses Gemini 2.5 Flash as its AI Chatbot which allows Cyber Analysts to interact with the chatbot. It also includes Gemini API Key which needs to be entered by the user so it helps to access the Threat Graph and the AI Chatbot for communication purposes. There are some shortcuts includes summarizing the whole pdf/docx file which makes it easier for users or professional Cyber Analysts to understand the context of the file in a much precise manner. Another shortcut known as Incident Ticket helps to generate a complete report on the vulnerabilities mentioned in the file making it easier once again to view incidents for security professionals. Some number tags in the answer by the chatbot will help the user to track the content taken in real time preview. Last feature which is included is the Audio-Briefing, this feature helps to read out the answers given by the chatbot, user can even edit the speed at what it needs to be played (1x,1.25,1.5x,1.75x). Audio-Briefing can be used to help read out incident tickets for the specific reports uploaded by the user. 

Finally it uses CLI in order to carry out small tasks such as doing short calculations or printing out the hexa values for colors being entered by the user. 

Overall, ThreatSynth is going to be used by Cyber Analysts in future in order to make incident reports easier to solve and figuring out sophisticated vulnerabilities with the help of this technology.
